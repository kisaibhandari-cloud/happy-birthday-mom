# 🎂 Happy Birthday Mom

A modern, interactive birthday webpage made with love for my mom, **Meltem**.

This project is a celebratory one-page experience featuring:
- a splash intro
- animated confetti and floating hearts
- a birthday gallery
- a personal message section
- an interactive cake candle moment
- background music
- a final celebration screen

## ✨ Features

- Elegant birthday landing page
- Smooth animations and festive effects
- Interactive cake / candle celebration
- Music toggle button
- Photo gallery section
- Personalized birthday message
- Mobile-friendly responsive layout
